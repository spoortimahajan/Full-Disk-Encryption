# 1. Introduction
Full Disk Encryption (FDE) is a security method used to protect all data stored on a disk by converting it into unreadable form. Only authorized users with the correct password or key can access the data. In this project, we created a simple web interface using HTML to explain and demonstrate FDE concepts.

# 2. Objectives
- To understand the concept of Full Disk Encryption
- To create a basic web page explaining FDE
- To show how encryption protects data
- To demonstrate tools like BitLocker and VeraCrypt
- To improve knowledge of HTML, CSS, and basic backend
  
# 3. Tools and Technologies
- HTML → for creating web pages
- CSS → for styling
- Python (Flask) → for backend (app.py)

# 4. Implementation
- Created project folders: static, templates, Screenshots
- Designed webpage using HTML (index.html)
- Added styling using CSS (style.css)
- Used Flask (app.py) to run the web app
- Displayed information about FDE and its working
  
# 5. Working Procedure
- Users enters password in web page
- Java script sends it to flask backend
- Flask verifies the password
- Result is shown on screen
- If correct--system unlocks
- If wrong--error message shown
  
# 6. Output
- The system displays a login screen and shows success or error message based on password verification
  
# 7. What I Learned
- Basics of Full Disk Encryption
- How encryption secures data
- HTML and CSS web development
- Basics of Flask (Python backend)

# 8. Conclusion
This project helped in understanding the importance of data security using Full Disk Encryption.
